# YaCut — сервис для сокращения ссылок

**YaCut** — это простой и удобный сервис для сокращения длинных URL.  
Написан на Python с использованием фреймворка Flask, SQLAlchemy и Jinja2.

---

## Возможности

- Сокращение длинных ссылок в один клик
- Возможность задать собственный короткий идентификатор (например, `ya.ru/mylink`)
- Автоматическая генерация уникального ID, если пользователь не указал свой
- Защита от дубликатов и использования недопустимых символов
- Загрузка файлов и создание коротких ссылок на них (интеграция с Яндекс.Диск)
- Адаптивный интерфейс на Bootstrap

---

## Технологии

- **Python 3.9+**
- **Flask** — веб-фреймворк
- **SQLAlchemy** — ORM для работы с базой данных
- **SQLite / PostgreSQL** — хранилище данных
- **Bootstrap 4** — стилизация интерфейса
- **Jinja2** — шаблонизация
- **pytest** — тестирование
- **aiohttp** — асинхронные запросы к API Яндекс.Диска

## Установка и запуск

1. Клонируйте репозиторий:
   ```
   bash
   git clone https://github.com/ваш-логин/async-yacut.git
   cd async-yacut
   ```
## Создайте и активируйте виртуальное окружение:

```
bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate     # Windows
```

## Установите зависимости:
```
bash
pip install -r requirements.txt
```

## Настройте переменные окружения (в файле .env, если используется):
```
env
DATABASE_URI=sqlite:///db.sqlite3
SECRET_KEY=ваш_секретный_ключ
```

##Запустите приложение:
```
bash
python run.py
```
## Перейдите в браузере по адресу: http://localhost:5000
